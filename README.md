# Nites Studios - Game Promotion Website

## Overview

This project was developed for **Nites Studios**, a game development company preparing to launch an upcoming title. The website was built using **Django** and serves as a promotional platform for the game, as well as an informational hub about the company and its team.

The main objective of the project is to generate awareness and engagement for the upcoming game release.

---

## Features

### 🎮 Game Promotion
- Dedicated section to showcase the upcoming game
- Promotional content including descriptions, visuals, and announcements
- Space for trailers, screenshots, and release updates

### 👥 About Us
- Company presentation page
- Background history of Nites Studios employees
- Professional experience and contributions of the development team

### 🧩 Django Backend
- Built with Django framework
- Organized structure following best practices
- Scalable architecture for future improvements
